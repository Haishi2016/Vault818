# Test intructions

1. Open a Windows PowerShell window.
2. Import the module

```powershell
Import-Module /path/to/HelloWorldModule.psd1
```

2. Invoke method
```powershell
Show-HelloWorld
```
3. The above command should generate output like this:
```powershell
Hello, World! Today is 09/28/2017 00:00:00.
```
